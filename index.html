Tableau de bord Power BI - Analyse Alimentaire

Description

Projet réalisé en groupe de 5

Conception et développement d'un tableau de bord Power BI dédié à l'analyse de produits alimentaires. Le projet vise à classifier les produits selon leur composition nutritionnelle (sucre, additifs, matières grasses, etc.) pour offrir une vision claire de leur qualité sanitaire et aider à la décision.

Moteur de recherche et de recommandation de séries TV

Description

Projet réalisé en groupe de 2

Conception et développement d'un moteur de recherche et d'un système de recommandation de séries TV à partir de fichiers de sous-titres. Le projet comprend un travail significatif en TAL (TF-IDF, tokenization) pour structurer les données, une base de données PostgreSQL pour l'indexation, et une interface graphique Tkinter. Tout a été déployé sur une machine virtuelle Linux.

Galerie du projet

Technologies Utilisées

  • Python (TAL, Tkinter)
  • PostgreSQL
  • TF-IDF
  • Linux

Compétences utilisées

Gerer Conduire Collaborer

Fonctionnalités

  • Recherche sémantique dans les sous-titres
  • Système de recommandation basé sur les préférences
  • Gestion haute performance de base de données
  • Déploiement sur machine virtuelle Linux
